1) При истановки программы python-3.6.2-amd64, отметте галочку Add Python 3.6 to PATH
2) После полного завершения установки введите в поиске CMD и откройте
4) В командной строке напишите pip install pygame и нажмите Enter
5) Подождите пока всё установится и командная строка попроси ввод как при первом запуске
6) В командной строке напишите pip install cryptography и нажмите Enter
7) Подождите пока всё установится и командная строка попроси ввод как при первом запуске
8) Закройте командную строку
9) Зайдите в папку с игрой Alien Invasion
10) Файл alien_invasion предназначен для запуска игры
11) Щёлкните по нему правой кнопкой, создать ярлык, и перетащите ярлык в удобное для васместо (рабочий стол)
12) Тепер вы можете запускать игру с этого ярлыка
